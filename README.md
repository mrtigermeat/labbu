"labbu" is a tool created to assist in contextual editing of phonetic "lab" files. These files are common place in a lot of SVS tools. (DiffSinger, NNSVS, etc)
# Installation

`pip install labbu`
# Quick Usage

```python
import labbu

labu = labbu.labbu()
labu.load("sample/sample.lab")

labu.check_label()
```
